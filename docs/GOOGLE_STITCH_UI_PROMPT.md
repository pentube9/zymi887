Create a separate prompt for Google Stitch to design ZYMI chat UI.

Goal:
Design a premium WhatsApp-inspired but original real-time communication app UI for ZYMI.

Design Requirements:
1. Desktop layout:
   - Left sidebar with user list
   - Search bar
   - Online/offline indicator
   - Last message preview
   - Unread badge
   - Main chat panel
   - Chat header with avatar, name, status, audio call, video call, more menu
   - Message area with sent/received bubbles
   - Typing indicator inside chat area
   - Composer at bottom with emoji, attach, location, mic, send

2. Mobile layout:
   - Full-screen chat list
   - Full-screen chat conversation
   - Back button in chat header
   - Bottom composer safe-area support
   - Touch target minimum 44px
   - No horizontal overflow

3. Calling UI:
   - Incoming call modal
   - Audio call overlay
   - Video call overlay
   - Local video small window
   - Remote video full area
   - End call, mute, camera toggle buttons
   - Calling states: ringing, connecting, connected, reconnecting, ended, failed

4. Media messages:
   - Image preview bubble
   - Video player bubble
   - Document card
   - Location card
   - Upload progress
   - Failed upload retry state

5. Visual Style:
   - Modern WhatsApp-inspired
   - Premium glass/soft shadow
   - Rounded message bubbles
   - Clean typography
   - Consistent spacing
   - Dark mode friendly
   - Professional, scalable, not childish

6. Do NOT copy WhatsApp exactly.
   Create a unique ZYMI design language inspired by WhatsApp UX.

Output from Google Stitch must include:
- Desktop screen
- Mobile chat list screen
- Mobile chat conversation screen
- Video call screen
- Incoming call modal
- Media message examples
- Design tokens: color, spacing, radius, shadow, typography
